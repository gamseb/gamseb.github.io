2019-12-16

Creating animated wallpapers from .gif or video files

There is one thing that I wanted to do for a long time. I was a bit envious of the wallpaper engine from windows. that allows you to run a gif/video in the background as wallpaper. So I have decided to create a script with similar functionality in i3wm. This is how I did it:

Prerequisites:

  • xwinwrap

animated_background.sh

#!/bin/sh

[ -z "$1" ] && exit

killall xwinwrap

sleep 0.3

xwinwrap -ov -g 1366x768+0+0 -- mpv -wid WID "$1" --no-osc --no-osd-bar --loop-file --player-operation-mode=cplayer --no-audio --panscan=1.0 --no-keepaspect --no-input-default-bindings --hwdec&

Make the file executable chmod +x animated_background.sh

To run the script simply run ./animated_background.sh <path-to-gif/video> from the file location.
