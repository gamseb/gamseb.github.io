Hosting hugo on github

2019-12-16

Hugo is a static website generator and github pages are a free way to host webpages

Prerequisites:

  • Hugo (https://github.com/gohugoio/hugo)
  • A working hugo website

Hosting on github

  1. Create two github repositories:

    One will contain the hugo source files and can be named anything you like. I will name it blog.

    The second one must be called .github.io and will contain the final files.

  2. Clone the them to your computer and put them both into one folder.

  3. Copy your hugo files into the first repository and make sure it works with hugo serve.

  4. Once ready to deploy, commit your changes. Then use hugo -d ../<username>.github.io to deploy the websites files to the folder. By using this command you will get the files in the format that is needed to host them on github pages.

  5. cd into the other folder and use git add *, then git commit -m "<message>" and then git push. You will be asked for your password and username.

  6. Once done you can navigate to .github.io to see the webpage.
